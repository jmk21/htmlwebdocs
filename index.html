Hello, World! - HTML5

Today we will be learning how to make a "Hello, World!" program with HTML. There is only one simple line of code, as listed below.

<p>Hello, World!</p>

And that's it! One single line and it's done.

Hello, World!

Our next lesson is using the <style> tag to make your webpage look better.

Using the style tag - CSS3

This lesson we will be learning how to use the <style> tag. The style tag helps us make our webpage look better.

First, we are going to color a paragraph based on a css 'rule' a rule is a line of code that defines how a element looks. Below is our rule for our <p> element.

<style>
p{
background-color: red;
}
</style>
<p>Hello, World!</p>

Below is our output.

Hello, World!

Our next lesson is about css psuedo classes.

Using psuedo classes - CSS3

Now we will learn how to use css psuedo classes. Psuedo classes are css callbacks that let us style an element by using a simple call back

Here is an example of a :hover css callback.

A hover callback activates when your mouse is hovering over the element.

<style>
p{
background-color: white;
}
p:hover{
background-color: red;
}
</style>
<p>Hello, World!</p>

Our output is below.

Hello, World!
